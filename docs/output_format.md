## 十、标准输出格式

Agent 回答研究问题时，应优先使用以下结构。

### 1. 问题理解

用 2 到 5 句话说明用户问题的本质，明确研究对象、时间范围和输出目标。

### 2. 已确认事实

列出已经验证的信息，包括：

- 数据或事件
- 数值或方向
- 时间
- 来源类型
- 是否为最新数据

### 3. 关键驱动因素

按重要性排序列出当前最重要的影响因素。

建议分类：

- 供给
- 需求
- 库存
- 价差
- 汇率
- 宏观
- 政策
- 情绪和资金

### 4. 判断结论

输出明确结论。

建议格式：

- 对 SC 的影响：利多 / 利空 / 中性
- 影响周期：短期 / 中期 / 长期
- 影响强度：强 / 中 / 弱
- 判断概率：高 / 中 / 低

### 5. 依据

按重要性排序列出核心证据。

每条依据应包含：

- 证据内容
- 来源
- 时间
- 这条证据如何影响结论

### 6. 替代解释

列出可能存在的其他解释，避免只给出单一判断。

### 7. 风险与不确定性

列出主要风险，例如：

- 数据缺口
- 时效性风险
- 地缘政治风险
- OPEC+ 政策风险
- 汇率风险
- 库存统计口径差异
- 国内政策变化
- 外盘和内盘交易时段错位

### 8. 下一步验证方向

列出后续需要继续跟踪的指标或事件。

例如：

- SC 主力价格和持仓变化
- SC 月差变化
- Brent、WTI、Oman、Dubai 价格
- 人民币汇率
- EIA 周度库存
- OPEC+ 会议或月报
- 中国原油进口和加工量
- 国内炼厂开工和成品油需求
- 交易所仓单和交割公告

### 9. 置信度

给出置信度：

- 高
- 中
- 低

并说明原因。
